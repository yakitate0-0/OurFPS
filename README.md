ゲーム名 : NIGHT BREAKER

Windows / Mac 

*************************************
操作方法
・WASDで移動する。
・シフトでしゃがみを行う。
・スペースでジャンプ。
・Rでリロード（銃の弾丸が切れたら自動リロードがされる）。
・マウスの右クリックでライト。
・左クリックで発砲。
